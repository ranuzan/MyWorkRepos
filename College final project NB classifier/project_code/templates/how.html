How it works

Naïve bayes classifiers

Naïve bayes is a simple statistic yet highly scalable classifier that relies on bayes theorem its advantage is that it only requires a small amount of training data to estimate the parameters necessary for classification.

the process of building our classifier is as followes:

• a data set is fed to the classifier with examples of text and their class

• the data set will be broken down and represented as vectors of feature values

• given a new example spacy (a syntactic dependency parser) will break the text to sentences

•the classifier will break these sentences into vectors and predict the class(offensive, nonoffensive) of each based on its features and values that match the data set that was fed onto it

to test our classifier we used a system called 10-fold cross validation we trained our algorithem with 90% of our data set and than tested its output on the remaining 10% the results can be seen in the attached excel file
